## BEM 작명 규칙
- __ : ~의 일부분
- -- : ~의 상태
- - : 일반적인 사용

```html
<div class="body__container"></div>
<div class="btn--success"></div>
<div class="toggle-btn"></div>
```

## Vendor Prefix
- 브라우저 업체별 접두사
- 실험적인 기능을 사용할 수 있도록 한다.
- -webkit- : 크롬
- -ms- : 익스프로러, 엣지 
- -moz- : 파이어폭스
- -o- : 오페라
